# Reporting and Dashboard Design

## 1. Bao cao quan tri

- Sales pipeline theo kenh, vung, sales rep
- Quotation conversion rate
- Doanh thu thuc hien so voi target
- OTIF giao hang
- Fill rate theo kho
- Near-expiry inventory
- Slow-moving/dead stock
- AR aging
- AP aging
- Gross margin theo item, channel, customer
- Batch release turnaround time
- Reject rate theo supplier
- Production yield variance
- PnL theo branch va cost center

## 2. Dashboard theo role

### CEO/GM

- Revenue MTD/YTD
- Gross margin
- Inventory days
- Cash position
- OTIF

### Sales Manager

- Pipeline
- Booking
- Top customers
- Overdue AR

### Warehouse Manager

- Inbound/outbound volume
- Picking productivity
- Fill rate
- Near-expiry

### QA Manager

- Pending inspections
- Hold batches
- Temperature excursions
- Recall status

### Finance Manager

- AR/AP aging
- Bank balances
- Credit exposure
- Tax status

## 3. Ky thuat bao cao

- Bao cao operational doc truc tiep tu ERPNext.
- Bao cao quan tri nang co the extract qua data warehouse/BI.
- Job batch dem de pre-aggregate KPI nang.
